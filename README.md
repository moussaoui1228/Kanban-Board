## dragstart event

dragstart is an event that fires when the user starts dragging a draggable element.

Imagine dragging a file on your desktop.

In a webpage, JavaScript can detect that moment.

### A complete drag-and-drop flow

There are several drag-related events:

dragstart	   User starts dragging
Event	         When it happens
dragenter	   Draggable enters a drop area
drag	       While the element is being dragged
dragover	   Draggable is over a drop area
dragleave	   Draggable leaves a drop area
drop	        User releases the element on a drop area
dragend     	   Dragging finishes
